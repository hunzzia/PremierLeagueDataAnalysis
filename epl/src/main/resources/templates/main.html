OFF THE BALL

We love soccer!

이 프로젝트는 England PremierLeague 경기 데이터를 이용하여 여러 가지 정보를 출력하는 프로잭트 입니다.
사용자와 관리자로 권한을 나누어 유저에 따른 권한을 달리해 서로 상호작용을 하는 프로잭트 입니다.
모든 유저는 실시간으로 경기 상황을 볼 수 있으며 관리자는 실시간의 경기 데이터를 저장해 사용자에게 서비스를 제공 합니다.


실시간 경기


관리자가 해당 경기에 참가

실시간으로 경기 정보를 입력

유저들은 경기 관람을 못해도 결과를 알 수 있다!

상세 정보


관리자는 팀의 정보와 선수의 정보를 입력

그에 따른 통계 차트 출력

사용자는 출력된 통계와 선수의 정보를 볼 수 있다!

게시판 기능


관리자는 공지사항 밑 게시판 관리

사용자는 쓰고자 한 내용을 게시판에 작성

사용자는 해당 게시판에 대한 자신의 의견 댓글 작성가능
